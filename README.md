# Frontend de Rebuscada

Aquest frontend s'ha creat amb [Create React App](https://github.com/facebook/create-react-app).

## Scripts disponibles

Des del directori del frontend podeu executar:

### npm start

Inicia l'aplicació en mode de desenvolupament.
Obriu [http://localhost:3000](http://localhost:3000) al navegador.

La pàgina es recarrega automàticament quan feu canvis.
Si hi ha errors de lint, es mostren a la consola.

### npm test

Inicia el runner de tests en mode interactiu.
Podeu consultar més informació a la documentació de [running tests](https://facebook.github.io/create-react-app/docs/running-tests).

### npm run build

Genera la versió de producció a la carpeta build.
Empaqueta React en mode de producció i optimitza el rendiment.

La sortida es minifica i els noms de fitxer inclouen hash.

### npm run eject

Atenció: aquesta acció és irreversible.

Aquest comandament copia tota la configuració (webpack, Babel, ESLint, etc.) dins del projecte perquè la pugueu controlar directament.
